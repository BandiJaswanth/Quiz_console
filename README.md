# 🧠 Quiz Application (Java Console)

A simple console-based Quiz Application built using Java that allows users to answer multiple-choice questions and receive an instant score based on their performance.

---

## 📌 Features

- Multiple-choice question system
- Score calculation based on correct answers
- Instant result display at the end
- Simple and interactive console interface
- Easy to extend with more questions

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Scanner class for user input
- Conditional statements and loops

---

## ▶️ How to Run

1. Open project in Eclipse / IntelliJ IDEA
2. Compile the Java file
3. Run the program
4. Answer the quiz questions in the console
5. View final score at the end

---

## 🎯 Learning Outcomes

- Strengthened Java fundamentals
- Improved logical thinking and decision-making
- Practice with loops, conditions, and input handling
- Understanding of simple scoring systems

---

## 👨‍💻 Project Type

Console-based mini project for practicing core Java programming.
